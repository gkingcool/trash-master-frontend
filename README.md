# Trash Master Frontend

Smart waste management dashboard with route optimization.

## Features
- Admin Dashboard with KPIs
- Driver Management
- Route Planner
- Real-time tracking

## Tech Stack
- React + Vite
- React Router
- Axios
- Chart.js

## Setup
\`\`\`bash
npm install
npm run dev
\`\`\`

## Backend
Requires Spring Boot running on http://localhost:8080
